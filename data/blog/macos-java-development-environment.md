剛好最近電腦重新安裝系統了，紀錄一下自已如何配置 Java 開發環境，安裝主要使用 [Homebrew](https://brew.sh/) 進行

```bash
# 個人習慣用 IntelliJ IDEA
brew install --cask intellij-idea-ce

# 根據自己專案安裝 JDK 版本，這邊推薦 openjdk@8 或 openjdk@11 相容性及穩定性較高
brew install openjdk@8

# 建立軟連結，否則系統認不到 JDK
sudo ln -sfn /usr/local/opt/openjdk@8/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-8.jdk

# 檢查 Java 版本
java -version
# openjdk version "1.8.0_345"
# OpenJDK Runtime Environment (build 1.8.0_345-bre_2022_08_04_23_35-b00)
# OpenJDK 64-Bit Server VM (build 25.345-b00, mixed mode)
```
