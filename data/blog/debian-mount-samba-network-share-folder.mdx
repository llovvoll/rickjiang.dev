紀錄一下如何在 Debian 掛載 SMB(Samba) 網路共享資料夾

```bash
$ apt update
$ apt install cifs-utils

$ mkdir -p /mnt/smbshare
$ mount -t cifs -o username=user,domain=local,password=pass //192.168.1.1/smbshare /mnt/smbshare

```
