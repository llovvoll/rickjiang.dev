記錄一下遇到的問題，當 SSH 連線遠端 Linux(Ubuntu) 伺服器使用 vi 編輯器時無法使用方向鍵，並且會產生 A、B、C、D 字元，解決的方式是在 `~/.vimrc` 加入關閉兼容模式設定

```:.vimrc
set nocompatible
```
